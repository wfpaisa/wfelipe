# Web wfelipe (wfelipe)

A web for the site of wfelipe.

## Install the dependencies

```bash
pnpm i
```

### Start the app in development mode (hot-code reloading, error reporting, etc.)

```bash
pnpm l
```

### Lint the files

```bash
pnpm lint
```

### Format the files

```bash
pnpm format
```

### Build the app for production

```bash
pnpm build
```
